# Bookmark Manager

Phase 1 portfolio project for the [Navigators Guild](https://github.com/Navigators-Guild) apprentice path. A single-page bookmark manager built with HTML, CSS, and JavaScript. No frameworks, no build step, no server — open `index.html` in a browser to run.

## Files

- `DESIGN.md` — design document (purpose, features, technology, interface, out of scope)
- `index.html` — the entire app: HTML, CSS, and JavaScript in one file
- `PROCESS.md` — build process documentation, including review-driven changes

## Running

Open `index.html` in any modern browser. Data persists in `localStorage` under the key `bookmark-manager:bookmarks`.

## Scope

This is the curriculum-spec version: the seven features defined in `DESIGN.md`, built in layers per chapter 05 of the apprentice onboarding guide. Adversarial review feedback has been addressed in subsequent commits.
